# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using vanilla HTML, CSS, and JavaScript.  
The project is based on the given design reference and focuses on matching the UI closely while keeping the code clean and easy to understand.

## Features

- Play Rock, Paper, Scissors against the computer
- Random computer choice on every round
- Win, lose, and tie result screens
- Score tracking for both player and computer
- Rules popup
- Final celebration screen after winning
- Responsive layout for different screen sizes

## Tech Stack

- HTML
- CSS
- JavaScript

No CSS framework or JavaScript framework has been used.
